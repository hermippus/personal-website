# Personal Website
### Tech stack:
* Nuxt JS
* Tailwind CSS

My simple personal website v1.1.
Demo: https://personal-website-ar7.pages.dev

## Requirements
* Node JS >= v23.4.0

## Build setup
```bash
# install dependencies
$ npm install

# dev run
$ npm run dev

# build for production
$ npm run build

# production preview run
$ npm run preview

# generate static project
$ npm run generate
```
